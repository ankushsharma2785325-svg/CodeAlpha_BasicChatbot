# Basic Chatbot

A simple rule-based chatbot built using Python.

## Features
- Responds to basic user inputs
- Greets users
- Answers simple questions
- Exits when user types "bye"

## Technologies Used
- Python
- Loops
- Conditional Statements

## Example

You: hello
Chatbot: Hi there!

You: how are you
Chatbot: I am fine, thank you!

You: bye
Chatbot: Goodbye!
